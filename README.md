## About
Provides a 72-hour XMLTV EPG for Rakuten NL's TV service, which is updated automatically every night at 2 am.

## Usage
Grab the XMLTV file from this link and paste it into your favorite IPTV client:
```
https://raw.githubusercontent.com/stefankn/rakuten-uk-epg/master/epg.xml
```
